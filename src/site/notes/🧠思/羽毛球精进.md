# 羽毛球精进
#思 #运动

- 羽毛球的学习是一个螺旋上升的过程，从练习到思考再实践再总结。对羽毛球的认识也在不断加深，认知不断深入，**指导实践**，同时再从实践中发现问题，增进认知，最终知行合一。
- 战术
	- 准备接球时，料其落点。
	- 击球时，料其未料。
- 步伐
- 动作合理性
- 手法
	- 接球
		- 抽球
- 速度！！！相对速度 控制落点（位置差）停顿（时间差）击球速度
- 握拍 食指、拇指捏紧 下三指虚握 持拍偏上
- 步伐
	- 主用并步 易于调整 简洁一致 启动速度快
	- 区分被动球 主动球
- 击球
	- 架拍
	- 引拍（蓄力）
	- 爆发瞬时击球 发力由下至上
	- 平衡手
- 控制
	- 避免中场高球
	- 挑对角后场球
	- 高远后场球
	- 后场软压
	- 后场劈吊
	- 前场分斜线
	- 搓球
- 启动
	- 跳起 落地 弹起
- 如影如声
- 先机
- 多拍稳定性 出球合理性
